The ``1-my_list`` module
========================

Using ``MyList``
----------------

First, we import the class:
    >>> MyList = __import__('1-my_list').MyList

Testing basic functionality with the example from the task:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Testing an empty list:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Testing with negative integers:
    >>> neg_list = MyList([-5, -1, -10, 0])
    >>> neg_list.print_sorted()
    [-10, -5, -1, 0]

Testing with a single element:
    >>> single_list = MyList([42])
    >>> single_list.print_sorted()
    [42]
